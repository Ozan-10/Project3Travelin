# 🌍 Project3Travelin - Tour Booking System

Modern bir tur rezervasyon uygulamasıdır. ASP.NET Core MVC ve MongoDB kullanılarak geliştirilmiştir.

## 🚀 Proje Özellikleri

- Tur listeleme
- Tur detay sayfası
- Rezervasyon oluşturma
- Admin paneli
- MongoDB ile veri yönetimi

## 🛠️ Kullanılan Teknolojiler

- ASP.NET Core MVC
- MongoDB
- AutoMapper
- Bootstrap
- HTML
- CSS
- JavaScript

## ⚙️ Kurulum

1. Repoyu klonlayın:

```bash
git clone https://github.com/Ozan-10/Project3Travelin.git
```

2. MongoDB bağlantı ayarlarını `appsettings.json` içinde düzenleyin.

3. Projeyi çalıştırın:

```bash
dotnet run
```

## 📌 Projenin Amacı

Bu proje, ASP.NET Core MVC ve MongoDB teknolojilerini öğrenmek ve gerçek bir rezervasyon senaryosu geliştirmek amacıyla oluşturulmuştur.
